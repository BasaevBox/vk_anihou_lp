## ✨ Основные возможности

### 🎭 Развлечения и RP
- **!аниме** — случайные аниме-арты из VK-групп (поддержка SFW и NSFW-режима)
- **RP-команды** — обнять, поцеловать, пнуть, убить, погладить и ещё 15+ действий
- **!шутка**, **!цитата**, **!рандом**, **!кто**, **!выбери**
- **!погода** (демо-режим)

### 🛠️ Удобные инструменты
- **!команда** — создание собственных кастомных команд
- **!qr** — генерация QR-кодов
- **!видео** — скачивание видео с YouTube прямо в VK
- **!автостатус** — анимированный статус с 8 разными анимациями

### ⚙️ Система управления
- **Система доверенных пользователей** (`!дов + @id` / `!дов - @id`)
- **Черный список** (`!чс`)
- **Логи команд** (`!логи`)
- **!токен** — безопасная отправка токена в избранное

### 📊 Информация
- `!помощь` — полный список команд
- `!инфо` — информация о боте и аптайме
- `!пинг`, `!время`
